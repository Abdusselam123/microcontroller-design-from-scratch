# microcontroller-design-from-scratch 
A step-by-step guide to designing 32-bit microcontroller architecture from gate-level to registers. For educational purposes.

🚀 About the Project
This project is a technical guide that explains step-by-step how a 32-bit microcontroller architecture is designed, starting from the most basic logic gates (gate-level) up to the register level. As a Mechanical Engineering student, I prepared this work to understand the "inner workings" of embedded systems and to bridge the gap between hardware and software.

This project is an improvised design process where new questions arise at every step, rather than a curriculum with strictly predefined boundaries. Therefore, the total number of chapters or the final scope has not yet been determined. New chapters will be added to the documentation as the design process evolves.
